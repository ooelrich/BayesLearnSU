# BayesLearnSU

This is a package containing data sets for the course Bayesian learning at Stockholm University. It contains the data sets

`campy`
`ebayBiddersData`
`rainfall`
`tempLinkoping`
`womenWork`

## Installation

Install using the `remotes` package

``` r
remotes::install_github("ooelrich/BayesLearnSU")
```
